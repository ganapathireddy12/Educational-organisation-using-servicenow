## 📌 Project Overview

### 🎯 Purpose

The main objective of this project is to **digitize educational workflows** using the ServiceNow platform.
It helps institutions manage **student admissions, academic records, and progress tracking** efficiently while reducing manual work.

---

## ✨ Key Features

| Feature                    | Description                                   |
| -------------------------- | --------------------------------------------- |
| Auto Admission Number      | Automatically generates unique admission ID   |
| PIN Code Address Auto-Fill | Fetches address based on PIN code             |
| Student Progress Tracking  | Calculates total marks, percentage and result |
| Admission Status Flow      | New → In Progress → Joined / Rejected         |
| Modular Design             | Separate modules for admin and teachers       |

## ⚙️ How to Run the Project

1️⃣ Create a ServiceNow Developer Account
👉 https://developer.servicenow.com

2️⃣ Request a **Personal Developer Instance (PDI)**

3️⃣ Navigate to
**System Update Sets → Retrieved Update Sets**

4️⃣ Import the update set from the project files

5️⃣ Click **Commit Update Set**

6️⃣ Access the modules:

* Admissions
* Student Progress
